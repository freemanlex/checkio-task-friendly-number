Можна скоротити довгі числа, щоб вони виглядали приємніше, тому давайте напишемо код, щоб зробити це.

Тобі слід написати функцію для перетворення number в рядок за допомогою декількох правил. Перш за все, тобі потрібно перерахувати число за заданою базою (base; за замовчуванням 1000). Значення є числом з плаваючою крапкою з числом десяткових знаків (decimals; за замовчуванням 0). Використовуй заокруглення до 0 (5.6 ⇒ 5, -5.6 ⇒ -5) якщо decimals = 0, в іншому випадку - стандартне заокруглення. Якщо кількість десятків перевищує поточну кількість цифр після крапки, доповни значення нулями. Число повинно бути значенням з літерами, що визначають степінь. Тобі буде дано список степеней (powers; за замовчуванням ['', 'k', 'M', 'G', 'T', 'P', 'E', 'Z', 'Y']). Якщо тобі дано суфікс (suffix; за замовчуванням ""), тобі слід його додати. Якщо позначень степеня не достатньо - залишайся на максимальному. І нуль це завжди нуль без степеня, але з суфіксом.

Сфера застосування: У фізиці, і в ІТ багато великих чисел. Іноді нам потрібно зробити їх більш простішими та легшими для читання. Коли ми говоримо про гігабайти, іноді не потрібно знати точну кількість байтів або кілобайт.

Передумови:

  • 1 < base ≤ 1032
  • -1032 < number ≤ 1032
  • 0 ≤ decimals ≤ 15
  • 0 < len(powers) ≤ 32
